Detailed Description

Модуль управления схемой предзаряда

Этот модуль реализует логику управления реле предзаряда для безопасного заряда входных конденсаторов преобразователя. Предзаряд предотвращает броски тока при включении высоковольтной шины.

Инициализация модуля предзаряда

Выполняет начальную настройку модуля:

  • Сбрасывает флаг завершения предзаряда
  • Настройка пина GPIO должна быть выполнена в CubeMX
Note
Пин GPIO должен быть предварительно сконфигурирован в CubeMX как выход push-pull с соответствующими настройками
Warning
Процедура предзаряда критична для безопасности системы

void prechargeStart ( void )

Запуск процедуры предзаряда

Выполняет процедуру предзаряда входных конденсаторов:

  1. Проверка, не была ли уже выполнена процедура
  2. Включение реле предзаряда
  3. Задержка 5 секунд для заряда конденсаторов через токоограничивающий резистор
  4. Отключение реле предзаряда
  5. Установка флага завершения
Note
Использует блокирующую задержку HAL_Delay на 5 секунд
Warning
Должна вызываться только при нулевом напряжении на входе
Note
В режиме TEST_UNITY функция выполняется без реальных операций с GPIO
