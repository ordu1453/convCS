Detailed Description

Модуль работы с CAN-шиной (FDCAN)

Этот модуль предоставляет функции для инициализации, отправки и приема сообщений по двум CAN-шинам (FDCAN1 и FDCAN2), а также восстановления после ошибок.

Инициализация CAN-модулей

Инициализация обеих шин FDCAN (FDCAN1 и FDCAN2)

Функция выполняет инициализацию двух CAN-шин (FDCAN1 и FDCAN2):

  1. Запуск CAN2 с задержкой после CAN1
  2. Настройка глобальных фильтров
  3. Активация прерываний
Note
Порядок запуска важен: сначала CAN1, затем с задержкой CAN2
Warning
При ошибке инициализации вызывается Error_Handler()

Публикация телеметрии по CAN.

Отправка телеметрии по CAN.

Отправляет телеметрийные данные системы с ID 0x123. Формат данных:

  • Байт 0: Состояние системы (SystemState_t)
  • Байт 1: Маска ошибок (младший байт)
  • Байты 2-3: Входное напряжение (int16_t)
  • Байты 4-5: Выходное напряжение (int16_t)
  • Байты 6-7: Выходной ток (int16_t)
Parameters
stateТекущее состояние системы
errorMaskМаска ошибок системы
sУказатель на структуру сенсорных значений
Note
Если модуль CAN не инициализирован, функция ничего не делает

Восстановление после состояния Bus-Off.

Выполняет процедуру восстановления CAN-контроллера после ошибки Bus-Off:

  1. Остановка контроллера
  2. Задержка 100 мс
  3. Повторный запуск контроллера
  4. Настройка фильтров
  5. Активация прерываний
Parameters
hfdcanУказатель на дескриптор FDCAN для восстановления
Note
Для полного сброса может потребоваться вызов MX_FDCANx_Init()
