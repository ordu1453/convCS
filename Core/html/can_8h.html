Detailed Description

Заголовочный файл модуля работы с CAN-шиной (FDCAN)

Этот файл содержит объявления функций и глобальных переменных для работы с двумя CAN-шинами (FDCAN1 и FDCAN2).

Проверка статуса CAN-шин

Проверяет состояние обеих CAN-шин на наличие ошибок Bus-Off. При обнаружении ошибки выполняет процедуру восстановления.

Note
Рекомендуется вызывать эту функцию периодически для мониторинга

Периодическая обработка CAN.

Функция вызывается периодически каждые CONTROL_DT_MS миллисекунд. Она отслеживает время и чередует отправку сообщений между FDCAN1 и FDCAN2.

Периодичность вызова определяется конфигурацией системы
