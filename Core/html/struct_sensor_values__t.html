Detailed Description

Структура для хранения значений датчиков

Содержит измеренные значения всех аналоговых датчиков системы. Все значения представлены в 32-битном формате с фиксированной точкой или как целые числа с масштабированием.

Note
Единицы измерения:
  • Токи: миллиамперы (мА) или с масштабированием
  • Напряжения: милливольты (мВ) или с масштабированием
Для преобразования в физические величины используйте коэффициенты из config.h
