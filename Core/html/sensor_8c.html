Detailed Description

Модуль работы с датчиками и фильтрацией данных

Этот модуль предоставляет функции для чтения, калибровки и фильтрации данных с аналоговых датчиков (напряжения, токи) с использованием фильтра Калмана.

void initKalmanFilters ( void )

Инициализация фильтров Калмана

Инициализирует все фильтры Калмана для каналов измерения с заданными параметрами:

  • q (шум процесса) = 0.001f - определяет скорость реакции на изменения
  • r (шум измерений) = 2.0f - определяет степень сглаживания
  • initial_value = 0.0f - начальное значение
  • initial_error = 1.0f - начальная ошибка
Note
Меньшее значение q дает более плавный отклик, меньшее значение r дает большее доверие измерениям

Вычисление и сохранение калибровочных коэффициентов

Запуск процедуры вычисления калибровочных коэффициентов

Выполняет процедуру калибровки датчиков:

  1. Отключает прерывания на время калибровки
  2. Выполняет многократные измерения для каждого канала
  3. Вычисляет средние значения смещений
  4. Сохраняет коэффициенты во FLASH память
Warning
Функция отключает прерывания на время выполнения калибровки
Note
Калибровка должна выполняться при нулевых входных сигналах

void SensorCalibration ( void )

Процедура калибровки датчиков

Выполняет автоматическую калибровку всех датчиков системы:

  1. Накопление большого количества измерений (1000 сэмплов)
  2. Фильтрация через вспомогательные фильтры Калмана
  3. Вычисление средних значений смещений
  4. Сохранение калибровочных коэффициентов во FLASH
Note
Используется 1000 сэмплов для каждого канала
Warning
Стирает и перезаписывает сектор FLASH памяти
Note
В режиме DEBUG выводит информацию об ошибках через UART

◆ sensorRead()

void sensorRead ( void )

Чтение и обработка данных с датчиков

Выполняет последовательное чтение всех 5 каналов ADC, преобразование сырых значений в физические величины и их фильтрацию через фильтры Калмана.

Порядок каналов ADC:

  • [0]: Входное напряжение
  • [1]: Выходное напряжение
  • [2]: Входной ток
  • [3]: Выходной ток
  • [4]: Ток дросселя
Note
Функция блокирующая, время выполнения зависит от скорости ADC
Warning
Отрицательные значения напряжений обрезаются до нуля
