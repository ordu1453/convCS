Detailed Description

Модуль диагностики и контроля состояния системы

Этот модуль предоставляет функции для проверки состояния системы, включая контроль напряжений, токов и состояния драйверов IGBT.

Author
ordum
Date
Oct 16, 2025
Note
Коды ошибок представляют собой битовую маску: 0x00 - нет ошибок 0x01 - только перегрузка напряжения 0x02 - только недогрузка напряжения 0x03 - перегрузка + недогрузка 0x04 - только перегрузка тока 0x05 - перегрузка напряжения + перегрузка тока 0x06 - недогрузка напряжения + перегрузка тока 0x08 - только ошибка драйвера IGBT 0x09 - перегрузка напряжения + ошибка драйвера 0x0A - недогрузка напряжения + ошибка драйвера 0x0C - перегрузка тока + ошибка драйвера 0x0F - все 4 ошибки одновременно

uint8_t diagCheck ( const SensorValues_t * sensorValues,
uint32_t * errorMask )

Проверка состояния системы на наличие ошибок

Выполняет комплексную диагностику системы по следующим направлениям:

  1. Контроль токов (входной, выходной, дросселя) на превышение максимальных значений
  2. Контроль напряжений (входного и выходного) на превышение максимальных значений
  3. Контроль напряжений на понижение ниже минимальных значений
  4. Проверка сигналов ошибок от драйверов IGBT
Parameters
sensorValuesУказатель на структуру с текущими значениями датчиков
errorMaskУказатель на переменную для сохранения битовой маски ошибок
Returns
Статус наличия ошибок
Return values
0Ошибок не обнаружено (или режим отладки активен)
1Обнаружена хотя бы одна ошибка
Note
В режиме DEBUG_MODE функция всегда возвращает 0 (без ошибок)
Функция обновляет битовую маску ошибок даже если они игнорируются в DEBUG_MODE
Warning
Проверка драйверов IGBT отключена в режиме тестирования (TEST_UNITY)
