Detailed Description

Основной модуль управления преобразователем

Этот модуль реализует главный цикл управления преобразователем, включая обработку состояний системы, диагностику, PID-регулирование и взаимодействие с периферийными модулями.

◆ converterInit()

void converterInit ( void )

Инициализация преобразователя

Выполняет начальную инициализацию всех подсистем преобразователя:

  1. Инициализация датчиков
  2. Инициализация диагностики
  3. Инициализация предзаряда
  4. Настройка ПИ-регулятора с параметрами ШИМ
  5. Отключение ШИМ до проверки безопасности
Note
Должна быть вызвана один раз при старте системы
Warning
ШИМ отключен до завершения инициализации

void converterProcess ( SystemState_t state)

Основной процесс управления преобразователем

Выполняет один цикл управления преобразователем:

  1. Получение данных с датчиков
  2. Проверка диагностики
  3. Обработка изменений режима работы
  4. Управление ШИМ в зависимости от состояния
  5. ПИ-регулирование в режимах заряда/разряда
  6. Управление предзарядом
  7. Публикация телеметрии по CAN
