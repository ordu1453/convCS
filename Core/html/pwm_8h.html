Заголовочный файл модуля управления ШИМ

Этот файл содержит объявления функций для управления широтно-импульсной модуляцией на таймере TIM1.

Инициализация модуля ШИМ

Выполняет начальную настройку модуля ШИМ. В текущей реализации может быть пустой функцией-заглушкой, так как настройка таймера выполняется в CubeMX.

Note
Настройка таймера TIM1 должна быть выполнена в CubeMX
