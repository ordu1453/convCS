Detailed Description

Модуль реализации фильтра Калмана для одномерного сигнала

Этот модуль предоставляет реализацию одномерного дискретного фильтра Калмана для сглаживания зашумленных измерений. Фильтр основан на простой модели постоянного состояния (constant state model).

Фильтрация одного измерения

Применяет один шаг фильтра Калмана к новому измерению. Алгоритм состоит из двух этапов:

  1. Этап предсказания (prediction): обновление оценки состояния и ковариации
  2. Этап обновления (update): коррекция на основе нового измерения

Математическая модель:

  • Модель состояния: xₖ = xₖ₋₁ (постоянное состояние)
  • Модель измерения: zₖ = xₖ + vₖ, где vₖ ~ N(0, r)
Parameters

Returns
Отфильтрованное значение
Note
Если kf == NULL, возвращает исходное измерение без фильтрации
Фильтр использует простую модель постоянного состояния

Инициализация фильтра Калмана

Устанавливает начальные параметры фильтра Калмана:

  • Шум процесса (q) - определяет, насколько быстро фильтр реагирует на изменения
  • Шум измерений (r) - определяет доверие к измерениям
  • Начальное состояние (initial_value) - начальная оценка сигнала
  • Начальная ошибка (initial_error) - начальная неопределенность оценки
Parameters
kfУказатель на структуру фильтра Калмана
qКовариация шума процесса (чем больше, тем быстрее реакция на изменения)
rКовариация шума измерений (чем больше, тем больше сглаживание)
initial_valueНачальное значение оценки состояния
initial_errorНачальная ковариация ошибки оценки
Note
Рекомендуемые значения: q = 0.001, r = 2.0 для плавного сглаживания
Warning
Если kf == NULL, функция ничего не делает
