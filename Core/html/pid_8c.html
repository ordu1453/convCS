Detailed Description

Модуль ПИ-регуляторов (одиночного и каскадного)

Этот модуль предоставляет реализацию пропорционально-интегральных регуляторов для систем управления. Включает как одиночный ПИ-регулятор, так и каскадную структуру (внешний контур по напряжению, внутренний по току).

Обновление каскадного ПИ-регулятора

Выполняет один шаг расчета каскадного регулятора:

  1. Внешний контур (напряжение) вычисляет задание по току
  2. Внутренний контур (ток) вычисляет выходное воздействие
Parameters
pi2Указатель на структуру каскадного регулятора
voltageSetЗаданное напряжение (уставка внешнего контура)
voltageMeasИзмеренное напряжение (обратная связь внешнего контура)
currentMeasИзмеренный ток (обратная связь внутреннего контура)
Returns
Выходной сигнал регулятора (например, скважность ШИМ)
Note
Выход внешнего контура является уставкой для внутреннего контура
Такая структура обеспечивает лучшую динамику и устойчивость

unsigned int piUpdate ( PIController_t * pi,
float setpoint,
float measurement )

Обновление одиночного ПИ-регулятора

Выполняет один шаг расчета ПИ-регулятора:

  1. Вычисление ошибки регулирования
  2. Интегрирование ошибки
  3. Применение антивиндапа (anti-windup)
  4. Расчет выходного сигнала
  5. Ограничение выходного сигнала
Parameters
piУказатель на структуру ПИ-регулятора
setpointЗаданное значение (уставка)
measurementИзмеренное значение (обратная связь)
Returns
Выходной сигнал регулятора (ограниченный)
Note
Реализует дискретную форму ПИ-регулятора: u(t) = Kp*e(t) + Ki*∫e(t)dt
Warning
При больших Ki и dt может возникнуть переполнение интегратора
