Detailed Description

Модуль работы с FLASH памятью

Этот модуль предоставляет функции для чтения, записи и стирания калибровочных данных во FLASH памяти микроконтроллера STM32G4.

HAL_StatusTypeDef Flash_Erase ( void )

Стирание пользовательской области FLASH.

Выполняет стирание одной страницы FLASH (2 КБ) по адресу FLASH_USER_START_ADDR. Используется для подготовки области к записи новых калибровочных данных.

Алгоритм:

  1. Определение номера страницы по адресу
  2. Отключение прерываний
  3. Разблокировка FLASH
  4. Стирание страницы
  5. Блокировка FLASH
  6. Включение прерываний
Returns
Статус операции
Return values
HAL_OKСтраница успешно стерта
HAL_ERRORОшибка стирания или проверки
Note
Размер страницы FLASH для STM32G4 составляет 2 КБ
Warning
Все данные в странице будут безвозвратно удалены

void Flash_ReadVars ( FlashVars_t * vars)

Чтение калибровочных данных из FLASH.

Считывает структуру FlashVars_t из FLASH памяти по заданному адресу. Используется при инициализации системы для загрузки калибровочных коэффициентов.

Parameters
varsУказатель на структуру для сохранения считанных данных
Note
Функция выполняет простое копирование данных из FLASH в RAM
Не проверяет валидность данных (например, с помощью контрольной суммы)
Для повышения надежности рекомендуется добавить проверку CRC

◆ Flash_WriteVars()

HAL_StatusTypeDef Flash_WriteVars ( const FlashVars_t * vars)

Запись калибровочных данных во FLASH.

Записывает структуру FlashVars_t во FLASH память с выравниванием на 8 байт. Используется для сохранения калибровочных коэффициентов после калибровки.

Алгоритм:

  1. Выравнивание данных на границу 8 байт
  2. Отключение прерываний
  3. Разблокировка FLASH
  4. Последовательная запись doubleword (64-bit) слов
  5. Блокировка FLASH
  6. Включение прерываний
Parameters
varsУказатель на структуру с калибровочными данными для записи
Returns
Статус операции
Return values
HAL_OKДанные успешно записаны
HAL_ERRORОшибка при записи данных
Note
Требует предварительного стирания страницы функцией Flash_Erase()
FLASH память поддерживает только запись 1→0, поэтому стирание обязательно
Warning
Максимальная частота записи FLASH ограничена (см. документацию STM32G4)
