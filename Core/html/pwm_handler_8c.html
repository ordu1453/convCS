Detailed Description

Модуль обработчика состояний ШИМ

Этот модуль реализует логику управления состоянием ШИМ в зависимости от текущего состояния системы и наличия ошибок. Определяет когда включать и отключать ШИМ на основе системных условий.

Обработка состояния ШИМ на основе системных условий

Обработка состояния ШИМ

Определяет необходимость включения/отключения ШИМ на основе:

  • Наличия ошибок диагностики
  • Текущего состояния системы
  • Запрошенного состояния перехода

Логика работы:

  1. При любой ошибке - ШИМ отключается, система переходит в INIT
  2. В состояниях INIT, IDLE, PRECHARGE - ШИМ отключен
  3. В состояниях CHARGE, DISCHARGE - ШИМ включен
Parameters
hasErrorФлаг наличия ошибок в системе (1 = есть ошибки, 0 = ошибок нет)
stateЗапрошенное состояние системы для перехода
Note
Функция обновляет глобальную переменную currentState
Warning
При обнаружении ошибок система принудительно переходит в STATE_INIT
Note
Состояние PRECHARGE считается безопасным для отключения ШИМ
