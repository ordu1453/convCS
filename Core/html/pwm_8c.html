Detailed Description

Модуль управления ШИМ (широтно-импульсной модуляцией)

Этот модуль предоставляет функции для управления ШИМ-сигналами на таймере TIM1 микроконтроллера STM32G4. Используется для управления силовыми ключами преобразователя.

void pwmDisable ( void )

Отключение ШИМ

Отключает генерацию ШИМ-сигналов на всех каналах. Используется для аварийного отключения или перехода в безопасное состояние.

Note
Все выходы переходят в неактивное состояние
Функция всегда выполняется, независимо от текущего состояния
Warning
После отключения ШИМ заполнение сохраняется и будет применено при следующем включении

void pwmEnable ( void )

Включение ШИМ

Включает генерацию ШИМ-сигналов на всех каналах:

  • Канал 1 (TIM_CHANNEL_1) - основной выход
  • Канал 1N (TIM_CHANNEL_1) - комплементарный выход
  • Канал 2 (TIM_CHANNEL_2) - основной выход
  • Канал 2N (TIM_CHANNEL_2) - комплементарный выход
Note
Функция имеет защиту от повторного включения
Используется для управления полумостовой/мостовой схемой
Warning
Включение ШИМ при ненулевом заполнении может вызвать броски тока

Получение значения регистра ARR таймера

Возвращает значение регистра авто-перезагрузки (Auto-Reload Register) таймера TIM1. Это значение определяет максимальное значение для параметра duty в функции pwmSetDuty().

Returns
Значение регистра ARR таймера TIM1
Note
ARR определяет период ШИМ-сигнала
See also
pwmSetDuty() для установки значения заполнения

void pwmSetDuty ( uint32_t duty)

Установка скважности ШИМ

Устанавливает значение заполнения (duty cycle) для каналов ШИМ. Поддерживает два канала с одинаковым значением заполнения.

Parameters
dutyЗначение заполнения (0..ARR)
Note
Если duty > ARR, значение ограничивается ARR
Устанавливает одинаковое значение для CH1 и CH2
Warning
Функция не проверяет состояние ШИМ (включен/выключен)
See also
pwmGetArr() для получения максимального значения
