void pwmHandlerProcess ( uint8_t hasError,
SystemState_t state )

Обработка состояния ШИМ

Определяет необходимость включения/отключения ШИМ на основе текущего состояния системы и наличия ошибок.

Parameters
hasErrorФлаг наличия ошибок в системе (1 = есть ошибки, 0 = ошибок нет)
stateЗапрошенное состояние системы для перехода
Note
Функция реализует приоритетную логику: ошибки > запрошенное состояние
